# Contributors

Tardigrade · Multi-GWAS Explorer was collaboratively developed by Amir Izadi and Zahra Aghabeygi, with distinct responsibilities across technical engineering and visual design.

## [Amir Izadi](https://github.com/amirizadi-1) — Technical Development, Software Engineering, and Deployment

* Designed and developed the application’s technical architecture and core functionality.
* Implemented the multi-GWAS analysis workflow, including data preprocessing, significant-gene extraction, gene-level overlap analysis, similarity calculation, consensus analysis, and report generation.
* Developed and integrated the application’s analytical visualization capabilities.
* Programmed the desktop application and integrated its analysis, visualization, reporting, and user-interface components.
* Optimized the application’s data-processing workflow, responsiveness, stability, and overall performance.
* Refactored and organized the codebase to support both the desktop application and the reusable Python library.
* Developed the programmatic Python API for accessing the project’s analysis, reporting, and visualization capabilities.
* Developed, packaged, documented, and published the `tardigrade-gwas` Python library on PyPI.
* Built and packaged the portable desktop releases for Windows and Linux.
* Implemented input validation, configurable analysis parameters, error handling, and safeguards for incomplete or invalid GWAS data.
* Performed software testing, debugging, troubleshooting, compatibility assessment, and technical refinement throughout development.
* Prepared the project for open-source publication, software distribution, and public release.

## [Zahra Aghabeygi](https://github.com/zahraaghabeygi) — Visual Identity, Branding, UI, and UX Design

* Designed the project’s overall visual identity and brand direction.
* Designed and selected the Tardigrade logo, including its adaptation for use on dark backgrounds.
* Designed the visual presentation and wordmark of the TARDIGRADE name.
* Developed and refined the project’s color system, including backgrounds, accent colors, interface states, borders, buttons, typography, and status indicators.
* Designed the overall structure and visual organization of the user interface.
* Designed the dashboard pages, layouts, cards, controls, navigation elements, tables, and interface components.
* Defined the visual hierarchy, spacing, sizing, typography, padding, and consistency of interface components.
* Directed the user-experience design and helped define the application’s interaction flow.
* Reviewed and refined the interface through multiple design iterations to improve readability, usability, consistency, and visual coherence.
* Supervised the visual-design process through the final version and ensured consistency across the logo, wordmark, color system, dashboard structure, and overall product identity.

## Shared Ideation and Concept Development

We contributed equally to the project’s ideation and concept development. Its functional, analytical, visual, and product directions were shaped through continued discussion, evaluation, and collaboration between both contributors.

## Use of AI-Assisted Tools

We used AI-assisted tools as supporting resources in parts of both the technical engineering and UI/UX design workflows. These tools helped us explore possible solutions, accelerate selected testing and iteration tasks, assist with parts of the coding process, and review certain technical and visual details.

The application’s core development and design work remained under our direct control. We reviewed, tested, modified, and adapted all assisted outputs to the specific requirements of Tardigrade, and no generated output was included in the final project without our direct evaluation and approval.
