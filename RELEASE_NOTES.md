# Tardigrade v1.0.1

The first public desktop release of **Tardigrade · Multi-GWAS Explorer**.

Tardigrade is an open-source application for comparing and analyzing gene-level signals across multiple GWAS datasets.

## Available editions

### Windows

* Portable Windows x64 build
* No separate Python installation required
* Extract the complete ZIP archive and run `Tardigrade.exe`

### Linux

* Portable Linux x86_64 build
* Requires glibc 2.38 or newer
* No separate Python installation required
* Extract the archive and run:

```bash
chmod +x Tardigrade
./Tardigrade
```

## Main capabilities

* Multi-dataset GWAS analysis
* Significant-gene extraction
* Gene-level overlap analysis
* Pairwise similarity calculation
* Consensus analysis
* Analytical visualization
* Report and table generation
* Manual export of reports and plots

## Default analysis columns

```text
P-VALUE
MAPPED_GENE
```

## Python package

The Python-library edition is available on PyPI:

https://pypi.org/project/tardigrade-gwas/

Installation:

```bash
pip install tardigrade-gwas
```
